Pipes |

replaceByPipe

Beskrivning: För att ersätta en bokstav etc.

               
                <p><b> {{ 'Adam' | replaceByPipe: 'A': ' ' }}</b></p>
Output

{{ 'Adam' | replaceByPipe: 'A': ' ' }}

callbackFilter

Beskrivning: För att filtrera värden i en iteration etc.

               
                  <li *ngFor="let person of personArray | callbackFilter: filterObj">
                   <b>{{ person.firstName }}</b>
                  </li>
Output
  • {{ person.firstName }}

    • defaultPipe

    Beskrivning: För att visa ett default värde om inget värde finns.

     
               
                  <p><b>{{ nullVar | defaultPipe: 'default value' }}</b></p>
    Output

    {{ nullVar | defaultPipe: 'default value' }}

    debounceInputPipe

    Beskrivning: Fördröjning x antal ms sekunder (default 500) innan nytt värde visas.

     
                 
                  <p><b>{{ userInput | debounceInputPipe: 1200 }}</b></p>
    Output


    {{ userInput | debounceInputPipe: 1200 }}

    Pagination component

    Beskrivning: Paginering, sidnumrering – dynamiskt numrering av sidor. Komponenten konsumerar en array av typen 'any' som den beräknar längden och skapar upp x antal sidor med nummer.

    Utan extra vy för sidnumrering

                     
                  <pagination [items]="itemsArray" 
                              [itemsPerPage]="10"   
                              (pageChange)="onPageChanged($event)"></pagination>

    Med extra vy för sidnumrering

                     
                  <pagination [items]="itemsArray"
                              [numberOfPagesView]="true"
                              [itemsPerPage]="10"   
                              (pageChange)="onPageChanged($event)"></pagination>
    Output

    Utan extra vy för sidnumrering

    {{item.name}}

    {{item.name}}
